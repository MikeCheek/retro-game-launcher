🔧 TROUBLESHOOTING

Problem: "Cannot connect to server"
Solution: 
  - Make sure npm start finished successfully
  - Check nothing is running on port 3333
  - Try http://localhost:3333 in fresh browser tab

Problem: "ROM upload fails"
Solution:
  - Check file format is supported
  - Verify file size < 100MB
  - Ensure sufficient disk space

Problem: "No games showing in library"
Solution:
  - Click [SELECT] to go to Library
  - Click [UPLOAD ROM] to add a game
  - Wait for upload to complete
